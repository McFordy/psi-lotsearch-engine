## Section: Environmental Setting (New South Wales)

Generate the Environmental Setting section (Section 3) for a NSW PSI report. This section covers topography, regional geology, regional hydrogeology, groundwater utilisation, and surface water hydrology.

### Interpretation Rules

Topography: Describe the general topographic setting using elevation data. Reference the elevation dataset and describe the local terrain.

Regional Geology: Use the geological unit name, map symbol, and description from the NSW geological mapping. Include the lithological description. If soils data is available, include soil type. If Acid Sulfate Soil (ASS) data is present, note the probability classification.

Regional Hydrogeology: Classify groundwater beneficial use using ASC NEPM 2013 Groundwater Investigation Levels (GILs):
- Fresh water: TDS <500 mg/L (drinking water quality)
- Marginal water: TDS 500-1,500 mg/L
- Brackish water: TDS 1,500-5,000 mg/L
- Saline water: TDS >5,000 mg/L
Reference the ASC NEPM 2013 Table 1A(3) and cite the TDS range and beneficial use classification.
Include groundwater depth information if available from the watertable depth dataset.

Groundwater Utilisation (NSW format): Report the total number of bores within the buffer. Generate a 5-bore summary table showing the 5 closest bores with: Bore ID, Distance (m), Direction, Total Depth (m), Purpose, and Standing Water Level (m) if available.

Surface Water Hydrology: Identify the nearest waterway/water body and its distance and direction from the Site.

Use these headings:

### 3.2 Regional Geology
